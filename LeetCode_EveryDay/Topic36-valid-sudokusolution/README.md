9*9的数独满足：
数字 1-9 在每一行只能出现一次。 
数字 1-9 在每一列只能出现一次。 
数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。 
***
关键是如何遍历子数独，如图，子数独行列对应i，j的关系为`row=i/3*3+j/3`，`col=i%3*3+j%3`，比如遍历第一行时i=0，j由0到8，分别对应第一个子数独由(0,0)到(2,2)的坐标
